"what is Unity Catalog in Databricks?":

"Unity Catalog is Databricks' centralized governance layer that introduces a three-level namespace — catalog, schema, and table. Before Unity Catalog, Databricks only had hive_metastore with a two-level schema-table structure and no cross-workspace governance. Unity Catalog adds fine-grained access control, data lineage, and the ability to share data across workspaces. In my project I used the workspace catalog with an imdb_final_project schema to organize all Bronze, Silver, and Gold tables under one logical namespace."
